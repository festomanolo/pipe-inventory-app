# Pipe Inventory App - Windows Portable Version

This is the Windows portable version of the Pipe Inventory App. This application allows you to manage inventory for pipe products and other hardware items.

## System Requirements

- Windows 10 or Windows 11 (64-bit)
- At least 4GB of RAM
- At least 200MB of free disk space

## Installation Instructions

Since this is a portable application, no installation is required. Just follow these steps:

1. Extract the executable file from the ZIP archive (if it was provided in a ZIP)
2. Place the `Pipe Inventory-Portable-1.4.0.exe` file in a location of your choice
3. Double-click the executable to run the application

## First-Time Setup

When you run the application for the first time:

1. The application will create a database file in the same directory as the executable
2. You can start adding inventory items immediately
3. The data will be stored locally on your computer

## Features

- Inventory management for pipe products and hardware items
- Product categorization and type filtering
- Stock level tracking with low-stock alerts
- Search functionality
- Data persistence using SQLite database

## Troubleshooting

If you encounter any issues:

1. **Application won't start**: Make sure you have the latest Windows updates installed
2. **Database errors**: Check if you have write permissions in the directory where the app is located
3. **Display issues**: Try running the app in compatibility mode for Windows 10

## Data Backup

To back up your data:

1. Locate the `pipe-inventory.db` file in the same directory as the executable
2. Copy this file to a safe location regularly
3. To restore data, replace the current database file with your backup
